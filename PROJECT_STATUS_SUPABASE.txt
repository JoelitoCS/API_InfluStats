✨ CAMBIOS PRINCIPALES (MongoDB → Supabase)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

❌ REMOVIDO                          ✅ AGREGADO
────────────────────────────────────────────────────────────────────
MongoDB (NoSQL)                     PostgreSQL via Supabase
Mongoose ORM                        Prisma ORM
models/User.js                      prisma/schema.prisma
findOne() queries                   prisma.user.findUnique()

✅ MANTIENE LAS VALIDACIONES
────────────────────────────────────────────────────────────────────
✓ Email único (verificado en BD)
✓ Contraseña 8+ caracteres
✓ Mayúsculas, minúsculas, números
✓ Encriptación bcryptjs
✓ JWT tokens 30 días
✓ Endpoints API idénticos
✓ Frontend sin cambios

❓ PREGUNTAS FRECUENTES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

P: ¿Es gratis Supabase?
R: Sí, tier gratuito incluye todo lo que necesitas para desarrollo.

P: ¿Debo instalar PostgreSQL localmente?
R: No, Supabase lo provee en la nube. Solo si quieres desarrollo offline.

P: ¿Mi código del frontend cambió?
R: No, el frontend sigue igual. Solo cambió el backend.

P: ¿Los endpoints son los mismos?
R: Sí, /api/auth/register y /api/auth/login funcionan igual.

P: ¿Qué pasó con los datos en MongoDB?
R: Si tenías datos, necesitas migrarlos manualmente (ver MIGRATION_*).

P: ¿Es más seguro PostgreSQL?
R: Sí, PostgreSQL es más robusto para datos relacionales.
